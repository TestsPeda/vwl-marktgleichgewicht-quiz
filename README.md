# Selbsttest — Vollkommener Markt & Preisbildung (Thema 11)

Eigenständige, statische Lern- und Test-Website zum Thema 11 **Preisbildung**. Sie deckt drei Bereiche ab:

1. **Der vollkommene Markt** – Markt, Marktformen, Modell-Annahmen, Gesetz von Angebot & Nachfrage
2. **Marktgleichgewicht ermitteln** – Schnittpunkt, Über­hänge, Tabelle / grafisch / mathematisch
3. **Verschiebung von Angebot & Nachfrage** – Ursachen, Folgen, Mindest- & Höchstpreis

Dazu eine **Lernseite** mit allen Kernbegriffen und Diagrammen zum Wiederholen.

## Nutzung

`index.html` im Browser öffnen — keine Installation, kein Build, keine Internetverbindung nötig.
Funktioniert offline und auf **GitHub Pages**. Multiple-Choice und Multi-Select werden automatisch
ausgewertet; Rechen-, Zuordnungs-, Freitext- und Szenario-Aufgaben haben eine aufdeckbare Musterlösung.

## Aufbau

```
index.html        Landing-Seite mit SVG-Navigation
01-…html          Test: Der vollkommene Markt
02-…html          Test: Marktgleichgewicht ermitteln
03-…html          Test: Verschiebung von Angebot & Nachfrage
lernseite.html    Konzepte + Diagramme (ohne Quiz)
assets/style.css  Layout & Design
assets/quiz.js    Interaktion (Auswertung, Lösungen)
```

## Umfang

42 Aufgaben · 8 inline-SVG-Diagramme · 28 Lernkonzepte.

## Hinweis zu Urheberschaft, Quellen & Haftung

Lernhilfe zu **reinem Bildungszweck.** Inhaltliche Richtigkeit ohne Gewähr. Keine Weitergabe als amtliches Prüfungsmaterial.
